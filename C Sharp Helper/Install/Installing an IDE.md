# Visual Studio Community Edition

This is the most common IDE used for C# due to it's amazing community support. You can use community edition as long as you have less than the specified computers at your workplace, which will easily be the case. 

**Within Visual Studio**
1. Download the .NET plugin `.NET desktop development`, you can also download other .NET plugins, but this one specifically have the Console App we need. 
2. Download the Unity plugin.
3. .NET - This should be a default install, but just confirm the box is ticked for the correct version before hitting install.

**External**
1. Download Git for Windows. 

| Key Notes    | Information                                                                                                                                                                                                    |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Console App  | When starting a new project, it is best to ensure that you use the Console App that is just called `Console App`. There is another version that says `.NET`, but this is a legacy version & it lacks features. |
| .NET Version | .NET versions are super important. If you are compiling using the latest .NET version & try to run this on a machine which isn't set up for it. Then you will likely not be able to run the executable.        |
